# grpc-contracts-artifacts

Auto-generated Go gRPC code from [proto-contracts](https://github.com/sososu-iwia/proto-contracts).

## Usage

```bash
go get github.com/sososu-iwia/grpc-contracts-artifacts@v1.0.0
```

## Packages

- `github.com/sososu-iwia/grpc-contracts-artifacts/payment` — PaymentService (ProcessPayment, ListPayments)
- `github.com/sososu-iwia/grpc-contracts-artifacts/order` — OrderService (SubscribeToOrderUpdates)

## Generation

Code is auto-generated via GitHub Actions in the [proto-contracts](https://github.com/sososu-iwia/proto-contracts) repository whenever `.proto` files are updated. Do **not** edit these files manually.
